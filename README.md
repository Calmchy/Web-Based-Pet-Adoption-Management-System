# 🐾 AdoptMe: Web-Based Pet Adoption Management System

> A centralized platform for managing pet adoption processes efficiently and securely.

---

## 📌 Overview

**AdoptMe** is a web-based system designed to help pet adoption centers streamline their operations. It provides a structured platform for managing pet listings, handling adoption applications, and tracking application status.

The system focuses on **digital pre-screening**, while ensuring that final adoption procedures are completed physically at the adoption center.

---

## 🎯 Purpose

The system aims to:

- Provide a **secure and centralized platform**
- Manage and maintain **pet listings**
- Handle **online adoption applications**
- Track and monitor **application status**
- Improve efficiency through **structured database management**

---

## 🌐 Scope

This system is designed for:

### 🏠 Adoption Center
- A specific pet adoption center or shelter

### 🛠️ Administrators
- Manage pet records (add, update, delete)
- Review adoption applications
- Approve or reject applications
- Update application status

### 👤 Registered Adopters
- Create and manage accounts
- Browse available pets
- Submit adoption applications online

---

## 🚧 System Boundaries

### ✅ Included Features
- User registration and authentication  
- Pet listing and management  
- Online adoption application submission  
- Application status tracking  

### ❌ Not Included
- Public pet marketplace  
- User-to-user pet listings  
- Full online adoption completion  

---

## 📍 Adoption Process Note

> ⚠️ This system only supports the **online pre-screening process**.

The following steps are completed **onsite at the adoption center**:

- Identity verification  
- Document signing  
- Final approval  
- Pet release  

---

## 🧱 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MariaDB  
- **Environment:** XAMPP
